## Installation Guide
### 1. Clone the Repository
```bash
git clone https://github.com/nipuran/FUTURE_ML_03.git
```
### 2. Navigate to the Project Directory
```bash
cd FUTURE_ML_03
```
### 3. Install Dependencies
```bash
conda env create -f environment.yml
```
### 4. Activate the Virtual Environment
- **Windows**:
  ```bash
  conda activate env
  ```
- **macOS/Linux**:
  ```bash
  conda activate env
  ```
